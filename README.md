# bot-translate

Translate text in another language

## Usage

```
Usage: translate [options] <language> <input...>

  Alias: tr

  Translate text in another language

  Options:

    --help  output usage information
```

## Config (config.yaml)

```yaml
translate:
  token: your_api_token
```

## How do I get my api token?

Go to https://tech.yandex.com/translate/
